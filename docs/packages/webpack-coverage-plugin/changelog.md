# webpack-coverage-plugin 更新日志

## 1.0.0

**发布日期**: 2024-12-18

### 新增功能

#### 核心功能
- 实现基础的 Webpack 插件框架
- 支持动态插桩功能
- 集成 babel-plugin-istanbul 实现代码覆盖率收集
- 提供 DevServer 集成，支持覆盖率数据上传接口

#### 配置选项
- `enabled`: 控制插件是否启用
- `include`: 指定需要插桩的文件模式
- `exclude`: 指定需要排除的文件模式
- `outputDir`: 指定输出目录

### 技术改进

- 使用 TypeScript 编写，提供完整的类型定义
- 遵循 Webpack 插件开发最佳实践
- 提供完整的单元测试覆盖
